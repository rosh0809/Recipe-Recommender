This project is a dynamic web application designed for contributing and browsing culinary recipes. It uses a robust backend architecture to handle data persistence between a web-based user interface and a relational database.
Add New Recipes: A dedicated interface to input recipe names, ingredients, and preparation instructions.

View All Recipes: Real-time retrieval and display of the entire recipe database.

MVC Architecture: Separation of concerns using Java Servlets (Controller), JSP (View), and JDBC (Model).

Database Connectivity: Centralized database management for reliable CRUD operations.

 Tech Stack
Language: Java

Web: Servlets, JSP, HTML/CSS

Database: MySQL

Server: Apache Tomcat (or any Servlet container)

Driver: MySQL Connector/J
